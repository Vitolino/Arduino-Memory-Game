# Arduino-Memory-Game
Lenguages: Portuguese(BR) and English

PORTUGUES:

Materiais necessários:
  1 Arduino
  1 Cabo USB A/B (cabo que vem junto com o arduino)
    Varios jumpers
  1 Protoboard
  4 LEDs
  4 Botões
  1 Display LCD 16x2
  1 Potênciometro
  3 resistores 330 ohms
  4 resistores 10k ohms
  
 Passo a passo:
 1º Faça a ligação dos componentes na protoboard de acordo com o esquemático;
 2º Baixe a IDE do arduino (https://www.arduino.cc/en/software)
 3º Abra o arquino .ino e carregue-o para seu arduino.
 4º Tudo pronto, agora só jogar.
