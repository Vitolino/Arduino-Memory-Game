# Arduino-Memory-Game


# NECESSARY MATERIALS:

  1 Arduino
  
  1 USB A / B cable (cable that comes with the arduino)
  
   Various jumpers
    
  1 Protoboard
  
  4 LEDs
  
  4 Buttons
  
  1 16x2 LCD display
  
  1 Potentiometer
  
  3 resistors 330 ohms
  
  4 resistors 10k ohms
